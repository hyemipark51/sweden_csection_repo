# Data dictionary

Cleaned CSV files derived from the official Socialstyrelsen 2023 appendix tables.

- `national_csection_trend_1973_2023.csv`: national long-term trend
- `regional_csection_2019_2023.csv`: regional trend, 2019-2023
- `regional_csection_detail_2023.csv`: regional total/emergency/planned detail for 2023
- `regional_deliveries_2023.csv`: deliveries and births by region in 2023
- `regional_csection_with_deliveries_2023.csv`: joined convenience file for 2023
